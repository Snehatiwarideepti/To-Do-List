# 📝 To-Do List App

A simple and user-friendly **To-Do List desktop application** built using **Java Swing**. The application helps users organize daily tasks, track completion status, manage deadlines, and improve productivity through an interactive graphical interface.

## ✨ Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Remove tasks
* ✅ Mark tasks as completed using checkboxes
* 📅 Assign optional due dates
* 🔍 Search tasks instantly
* 🔄 Sort tasks by:

* Name
* Due Date
* Completion Status
* 🚨 Automatically highlight overdue tasks
* 🎨 Clean and beginner-friendly graphical user interface

## 🛠️ Technologies Used

* Java
* Java Swing (GUI)
* AWT
* Object-Oriented Programming (OOP)

## 📋 Requirements

Before running the project, ensure you have:

* Java JDK 8 or higher
* Any Java IDE (IntelliJ IDEA, Eclipse, NetBeans, VS Code) or Command Prompt/Terminal

Verify installation:

```bash
java -version
javac -version
```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone <repository-url>
cd ToDoListApp
```

### Compile the Application

Navigate to the source directory and compile:

```bash
javac ToDoListApp.java
```

### Run the Application

```bash
java ToDoListApp
```

## 📖 How to Use

### Add a Task

1. Enter a task name in the input field.
2. Click **Add** or press **Enter**.

### Edit a Task

1. Double-click the task.
2. Update the details.
3. Save the changes.

### Complete a Task

* Click the checkbox beside a task to mark it as completed.

### Remove a Task

1. Select the task.
2. Click **Remove**.

### Search Tasks

* Use the search box to quickly find tasks by name.

### Sort Tasks

* Choose a sorting option from the dropdown menu:

  * Name
  * Due Date
  * Completion Status
